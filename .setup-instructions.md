# Foundation Setup Instructions

This document contains the remaining manual steps to complete the foundation setup.

## Step 1: Install Missing npm Dependencies

Run this command in the project root:

```bash
npm install framer-motion @react-three/fiber @react-three/drei three lenis shiki mixpanel-browser next-sitemap @types/three
```

## Step 2: Install shadcn/ui Components

Run these commands one by one:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add tabs
npx shadcn@latest add accordion
npx shadcn@latest add separator
```

## Step 3: Verify the Setup

Run these commands to ensure everything is working:

```bash
# Type check
npx tsc --noEmit

# Build check
npm run build

# Start development server
npm run dev
```
