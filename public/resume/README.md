# Resume Files

Place your resume files in this directory.

## Required Files

- `nadav-galili-resume.pdf` - PDF version of your resume
- `nadav-galili-resume.docx` - Word document version of your resume

## Optional Files

- `nadav-galili-cv.pdf` - Longer CV format if applicable
- `cover-letter.pdf` - Generic cover letter template

## Guidelines

- Keep file names URL-friendly (lowercase, hyphens instead of spaces)
- Ensure PDFs are optimized for web (smaller file size)
- Update resume files regularly to match the website content
- Include contact information in the resume files
