## Logo Guidelines

- Format: SVG preferred (scalable and small file size)
- If PNG: use transparent background, 256x256 or 512x512
- Ensure logos are licensed for use (most official logos are free to use)
- Consider using services like:
  - [Simple Icons](https://simpleicons.org/) - SVG icons for popular brands
  - [DevIcon](https://devicon.dev/) - Free icon set for programming languages and tools
  - [SVG Repo](https://www.svgrepo.com/) - Free SVG vectors
